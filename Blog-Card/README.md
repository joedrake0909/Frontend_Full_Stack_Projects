# Blog Card

A simple, responsive blog preview card built with HTML and CSS.

This project is a beginner-friendly frontend practice focused on layout, spacing, typography, and card styling.

## Preview

The card includes:
- Featured illustration
- Category tag
- Publish date
- Article title and description
- Author row with avatar and name

## Built With

- HTML5
- CSS3 (Flexbox)
- Figtree font files included in `assets/fonts`

## Project Structure

```text
Blog-Card/
|-- index.html
|-- style.css
|-- README.md
`-- assets/
    |-- images/
    |   |-- illustration-article.svg
    |   `-- image-avatar.webp
    `-- fonts/
```

## Getting Started

1. Clone or download this folder.
2. Open `index.html` in your browser.

No build tools or dependencies are required.

## Notes

- The page uses a centered card layout with a yellow background container.
- Styles are defined in `style.css`.
- Image and avatar assets are loaded from `assets/images`.
